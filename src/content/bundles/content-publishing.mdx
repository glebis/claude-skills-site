Two image generators cover different needs: GPT Image 2 for best-in-class text rendering, Nano Banana (Gemini) for fast cheap iteration. Brand Agency applies consistent neobrutalist styling across all artifacts. The presentation generator creates interactive HTML slide decks. PDF generation uses Pandoc with the Eisvogel template and EB Garamond for professional documents. Tufte Report builds standalone data dashboards with Chart.js and sparklines. Sketch opens a collaborative SVG canvas. Vision Bench scores and compares generated images using LLMs as judges.
