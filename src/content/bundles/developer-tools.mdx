TDD enforces red-green-refactor cycles using separate subagents for test writing and implementation — architecturally preventing the test author from seeing the implementation. LLM CLI pipes any file through any provider (OpenAI, Anthropic, Gemini, Ollama) with persistent config. Linear manages issues, projects, and workflows via a zero-dependency Python CLI. Session Finder uses semantic embeddings to search your entire Claude Code history and relaunch past sessions by topic. GitHub Gist publishes any file as a shareable snippet in one command.
