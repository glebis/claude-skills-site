## What it does

Creates interactive HTML presentations with neobrutalism styling and Agency brand colors. Generates from JSON/YAML content with smooth scrolling navigation, keyboard support, and multiple export formats.

## Output formats

- **HTML** — interactive presentation with navigation dots, keyboard arrows, smooth scrolling
- **PNG** — individual 1920x1080 slide images via Playwright
- **PDF** — multi-page document via Playwright

## Slide types

- **Title** — full-screen title with subtitle on a colored background
- **Content** — heading + body text + optional bullet list
- **Two-Column** — split layout for comparisons, text + image, before/after
- **Code** — dark background with syntax highlighting
- **Stats** — big numbers with labels ("14 templates | 4 formats | 1 skill")
- **Grid** — grid of cards with numbers, titles, descriptions
- **ASCII Art** — decorative slide with box-drawing characters

## When to use

When you need a styled presentation for a talk, pitch, or visual summary. Follows the Agency brand system automatically via the brand-agency skill.
