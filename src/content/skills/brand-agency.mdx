## What it does

Applies a consistent neobrutalism visual identity to any artifact Claude produces — presentations, SVG graphics, HTML documents, and web interfaces. Provides the complete brand system: color palette, typography stack, shadow and border styles, and application guidelines.

## Brand system

- **Colors** — primary orange (#e85d04), secondary yellow (#ffd60a), accent blue (#3a86ff), with chart green and red for data
- **Typography** — Geist ExtraBold for headings, EB Garamond for body text, Geist Mono for code
- **Style** — hard 4px offset shadows, 3px solid black borders, zero border radius, no gradients, high contrast

## When to use

Any time you're generating a visual artifact — presentation, SVG diagram, HTML report, dashboard — and want it to follow the Agency brand. Referenced automatically by other content-publishing skills like presentation-generator.
