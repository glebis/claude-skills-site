## What it does

Generates and edits images via OpenAI's GPT Image 2 API with an interactive, guided workflow. Excels at images with readable text — infographics, diagrams, posters, social media cards. Guides you through style selection, drafts cheap previews, then generates finals.

## Key features

- **27 style presets** — editorial, bauhaus, constellation, pixar, trading-card, action-figure, cinematic, nordic, flat-social, linkedin-hero, social-bold, and more
- **Draft-first workflow** — iterate at $0.006/image, upgrade to $0.21 for finals (35× savings)
- **Photo editing** — transform your photos into any preset style via multipart upload
- **Platform sizing** — auto-crops for YouTube, Instagram square, LinkedIn, slides, Pinterest
- **Carousel mode** — generate cohesive 10-slide stories for LinkedIn/Instagram
- **Cost confirmation** — prompts before spending >$0.50, shows estimates before every batch
- **Interactive guidance** — skill walks you through style, platform, draft→final flow step by step

## When to use

When you need generated images with readable text, social media carousels, LinkedIn quote cards, or want to transform photos into artistic styles. Use draft mode for cheap iteration, finals for publishing.
