## What it does

Walks a Claude Code Lab graduate through a four-part interactive retrospective that consolidates learning, captures best work, and plans next steps.

## Four parts

1. **Progress audit** — before/after skill comparison with a visual table showing growth across dimensions (CLI tools, coding, automation, APIs, etc.)
2. **Best prompt** — takes your most effective prompt from the lab and helps extract what made it work, turning it into a reusable pattern
3. **Monthly plan** — creates a concrete 30-day action plan based on your new skills and goals
4. **Feedback** — structured feedback collection (JSON + report) for the lab organizer

## When to use

At the end of a Claude Code Lab cohort, to reflect on what you learned and create a plan for continuing to build skills independently.
