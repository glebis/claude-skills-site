## What it does

Mimics the brain's default mode network by sampling 50 recency-weighted random note pairs from your vault, synthesizing connections via parallel Sonnet agents, then filtering for quality with a Haiku critic. Saves high-scoring insights as individual notes and a daily digest.

## Key features

- **Multi-agent architecture** — parallel Sonnet synthesis + Haiku quality filtering (score >= 7.0)
- **Recency-weighted sampling** — prioritizes notes modified in the last 120 days
- **Deduplication** — tracks sampled pairs in history log to avoid repetition
- **Rich output** — individual insight notes with wikilinks, daily digest with stats, daily note summary
- **Zero dependencies** — pure Claude Code tools (Glob, Read, Write, Bash, Task)

## When to use

When you want serendipitous connections between your notes without manually browsing — run it weekly or after adding substantial new content to your vault. Costs approximately $0.40-0.50 per run.
