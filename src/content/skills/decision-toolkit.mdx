## What it does

Creates interactive decision-support tools that help you think through significant choices systematically. Produces HTML artifacts — step-by-step guides, bias checkers, scenario explorers, and dashboards — that illuminate the decision space without deciding for you.

## Decision types

- **Opportunity evaluation** — should I pursue this partnership, job offer, investment, or project?
- **Resource allocation** — where should I invest my time, money, or attention?
- **Risk assessment** — what could go wrong and is it worth it?
- **Trade-off navigation** — which option among alternatives?

## Key principles

- **Cognitive inclusivity** — supports analytical (matrices, scores), intuitive (gut-check prompts), visual (diagrams, color coding), and sequential (step-by-step wizard) thinking styles
- **Accessible by default** — keyboard navigable, screen reader support, WCAG AA contrast, works without JavaScript
- **Nine-step journey** — context, values, options, evidence, risks, scenarios, biases, synthesis, action

## When to use

When facing any choice with multiple factors and real uncertainty — career decisions, partnership evaluations, technology selection, project prioritization.
