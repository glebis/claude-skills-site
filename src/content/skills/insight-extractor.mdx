## What it does

Reads the HTML report generated by Claude Code's `/insights` command and extracts six categories of actionable information: CLAUDE.md improvements, useful prompts and patterns, technical learnings, workflow improvements, tool discoveries, and automation candidates. Saves structured markdown to your Obsidian vault with MoC integration and daily note links.

## Key features

- **6 extraction categories** — action items, prompts/patterns, technical learnings, workflow improvements, tool discoveries, automation candidates
- **Three modes** — auto (saves everything), interactive (cherry-pick items), configure (set paths and preferences)
- **Vault integration** — insight files, MoC updates, daily note links, and task creation for automatable items
- **Multi-machine aware** — includes hostname in filenames, MoC groups by date across machines
- **Automation tagging** — candidates tagged #automation-candidate, agent-runnable ones tagged #agent-task

## When to use

After running `/insights` to persist the analysis into your vault, during weekly reviews to spot patterns, or when hunting for repetitive tasks worth automating.
