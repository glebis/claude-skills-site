## What it does

Searches and fetches emails from Gmail via the Gmail API. Supports flexible queries, multiple output formats, and attachment downloads — all from the terminal.

## Key features

- **Natural language queries** — find emails by sender, subject, date range, or keywords
- **Full message access** — read complete email bodies, not just subject lines
- **Attachment downloads** — save attachments directly from specific messages
- **OAuth authentication** — secure browser-based auth with automatic token refresh

## When to use

When you need to search Gmail, find specific emails, or download attachments without leaving the terminal. For broader Google Workspace operations (Calendar, Drive, Sheets), use the gws skill instead.
