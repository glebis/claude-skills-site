## What it does

Searches Chrome's local SQLite history database using natural language queries. Parses your intent, queries the database, filters out noise (social media, email, redirects), and groups results by type — reading, research, tools, events.

## Key features

- **Natural language** — understands "articles I read yesterday", "scientific articles about AI last week", "reddit threads last month"
- **Smart filtering** — automatically excludes social media, email, and utility sites to surface real content
- **Content clustering** — groups results into reading, research, tools, and events categories
- **Site-specific search** — filter by domain like "on medium" or "reddit threads"

## When to use

When you want to find something you browsed recently without scrolling through Chrome's history UI. Especially useful for recovering articles, research papers, or code repos you visited.
